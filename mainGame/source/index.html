Source Code & Setup

To set up this project locally:
1. Clone the repository:
git clone https://github.com/RayenAkrich/TypingGameWebsite.git
2. Set up a MySQL database using the provided SQL file in Connection/creationsSQL.sql.
3. Install Python dependencies:
pip install flask flask-cors mysql-connector-python
4. Run the Flask backend:
python Connection/signup/signup_server.py
5. Open mainGame/home/index.html in your browser to start the app.
